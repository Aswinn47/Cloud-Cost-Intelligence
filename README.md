## 📌 Project Overview

This is a **Final Year Undergraduate Project** built as part of the
Bachelor of Computer Applications (BCA) program.

Modern supply chain companies spend heavily on cloud infrastructure.
Without proper visibility, they overspend by 30–40% on services like
EC2, RDS, and Kubernetes clusters. This platform solves that problem by
automatically processing cloud billing CSV data and generating
intelligent cost reports.
